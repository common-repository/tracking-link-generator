Generate link for Analytics tool like Google Analytics and a short link.

== Description ==

Generates links for Analytics tools and short link. Enter your Campaign Name, Source, Medium (UTM link) to generate a full link and a short link (trough the bitly API) all in once.

Available languages :
* English

== Installation ==

1. Upload `tracking-link-generator` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
